Design, Prototype, and Implement BMS Algorithms with MATLAB/Simulink

Learn how to accurately simulate dynamic behavior of single battery cell using Simulink and scale your models all the way to perform system level simulation of entire battery pack. Use this dynamic models of battery pack to design and test algorithms for Battery Management System before generating production quality C/C++ code and targeting embedded processors.
This example Simulink Project includes:
  • Battery Cell Characterization
  • Detailed Battery Pack Plant Model
  • Designing BMS Algorithms
  • Code Generation
  • Verification & Validation (Requirements + Test Cases)
  • Hardware-In-Loop Testing of BMS
